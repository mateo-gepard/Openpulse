# Algorithm Specs

This directory contains the specification files for all 72 OpenPulse algorithms.

## How to create a new spec

```bash
# Copy the template
cp ../.agents/skills/openpulse_algorithm/templates/spec_template.md \
   A03_spo2/spec.md

# Fill in every field using A01_heart_rate as reference
```

## Directory Format

```
<ID>_<snake_case_name>/
├── spec.md          ← Algorithm specification (THE contract)
├── references.md    ← Extended literature references (optional)
└── test_vectors.h   ← Test data (generated by skill)
```
